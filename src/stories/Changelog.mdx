  Common formatters for hours and days have been added to the dayjs library.

- Temporary Components
  Note that the Plot and Table components are temporary and are specifically for use with chart items. They are not the public versions and may undergo changes in the future.

* Plot Component Usage Guide
  The Plot component is a complex component that uses Highcharts to display data in a variety of chart formats. It also includes a table view of the data and allows for interaction with the chart and table.

Props
The Plot component accepts the following props:

- data: An array of RowTable objects. Each RowTable object represents a row in the table and a series in the chart.
- optionsChart: A Highcharts.Options object that specifies the configuration for the chart.
- showTableChart: A boolean that determines whether the table view is shown initially.
- Tags: A function that returns a JSX element or null. This function receives an object with the following properties: isDialog, hideHeader, showTableCollapse, onChange.
